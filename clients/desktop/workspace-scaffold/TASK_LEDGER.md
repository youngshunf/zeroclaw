# TASK_LEDGER.md — 任务账本

_跨会话追踪你的工作。每次醒来，先看这里。_

## 使用规则

1. **接到复杂任务时**：在"当前任务"中添加一条记录
2. **委派给 sub-agent 后**：记录 task_id 和状态
3. **任务完成后**：更新状态为 ✅，写一句话总结，同时用 `memory_store` 存详细结果（key: `task:{主题}`）
4. **每次新会话开始**：读这个文件，检查是否有进行中的任务
5. **已完成任务**：移到下方"已完成"区域，只保留最近 10 条，更早的只保留在 memory 中

## 双轨记录

- **这个文件** = 仪表盘：只记概览（日期、任务名、状态、一句话总结）
- **memory_store** = 详情库：存完整结果、关键数据、可搜索的上下文

> 文件保持精简，防止膨胀。需要回顾细节时，用 `memory_recall` 搜索 `task:` 前缀。
