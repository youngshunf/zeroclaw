# _reference — OpenClaw 官方默认模板（中文翻译）

> 来源：OpenClaw 初始化 Agent 的默认工作区文件
> 原文：英文
> 翻译：原封不动的中文翻译，保留所有结构和语气

## 文件清单

| 文件 | 作用 |
|------|------|
| `AGENTS.md` | 工作区指南 — 每次会话怎么做、记忆怎么管、安全边界、群聊规则、心跳机制 |
| `SOUL.md` | 灵魂 — Agent 的核心行为准则和调性 |
| `IDENTITY.md` | 身份 — 名字、物种、调性、emoji（首次对话中填写） |
| `BOOTSTRAP.md` | 出生证明 — 首次启动流程，完成后删除 |
| `HEARTBEAT.md` | 心跳配置 — 周期性检查任务 |
| `TOOLS.md` | 本地笔记 — 环境特定的配置信息 |
| `USER.md` | 人类画像 — 关于主人的信息 |

## 用途

作为唤星 AI 模板设计的参考。对比我们的模板（`_base/` 和各模板目录），学习 OpenClaw 的设计理念。

## 关键设计理念

1. **对话式出生** — BOOTSTRAP 不是流水线，是一场对话。名字、调性、边界都是聊出来的。
2. **最小起步** — 文件初始都是空的模板，让 Agent 和人类一起填充。
3. **SOUL 是可进化的** — "这个文件属于你，由你来演化"，Agent 可以修改自己的灵魂。
4. **记忆分层** — 每日流水(memory/) + 长期精华(MEMORY.md)，明确区分。
5. **安全分级** — 对内大胆，对外谨慎。群聊中不代表主人发言。
6. **反废话** — "跳过'好问题！'直接帮"，核心态度贯穿始终。
